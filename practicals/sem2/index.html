Week 3

Summary

Week 3 was introducing us to Internet cookies - these are all about maintaining state on a webpage.

Cookies are small pieces of information that are:

  • Sent from a website
  • Stored on the visitors hard drive by the user's web browser

Cookies are usesd to differentiate among users and personalise websites They are used to:

  • Remember stateful information (e.g. items added into a shopping cart), manage the shopping cart, personalise aspects of websites
  • Record the user's browsing activity (e.g. clicking particular buttons, logging in, or recording which pages were visited in the past) - website statistics and creation of anonymous user profiles
  • Remember arbitrary pieces of information previously entered (e.g. names, addresses, passwords, and credit card numbers) - knows if the user is already logged in

Week 4 we began to look at the jQuery library, this is a fast and concise JavaScript Library that is designed to change the way that you write JavaScript

It offers some different functionality from plain JavaScript

$() function is a factory method that creates JQuery objects

Week 6

Summary

This week we looked at how to select all elements of a certain type, for example all img tags, or all elements with the same attribute.

This can be done through using
$(selector).each
And then you can apply a function to each of these elements through using
$(selector).each(function) {

For example, to get the attrivutes of all image alt tags you could use:
function getAlt(){
$("img").each(function(m){
alert(this.alt);
});
}

After this we looked at how you could select a subset of elements using the selector.slice(start, end) method.

This selects a subset of the matched elements. Starting from the 'start', to 'end' but excludes the 'end'.

For example:
$(document).ready(function() {
$("li").slice(2, 5).addClass("selected");
});
 Will add the class "selected" to the 2nd, 3rd and 4th list items.

    Week 9

    Summary

    This week we began to look at AJAX, which stands for Asynchronous JavaScript and XML which allows us to update pages without the need to refresh the page in the browser all the time. When using AJAX the thing to take in account is the XMLHttpRequest object which you can structure with data that you wish to send out, set data up with a call back function and connect to the server.

    The load() function loads data from a server and puts the returned data into the selected element

    					$(document).ready(function(){
						$("button").click(function(){
						  $("div").load('test1.txt');
						});
					  });

    The ready() function waits for the dom to be loaded, then runs the function. The click() function waits for the selected element to be clicked, in this case, a button. Finally the load() function loads the file test1.txt into the selected element "div" when the button is clicked, after the DOM is ready.

    					$(document).ready(function(){
						$("button").click(function(){
						  $.ajax({url:"test1.txt", success:function(result){
							$("div").html(result);
						  }});
						});
					  });
