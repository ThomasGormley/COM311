Week 3

Summary

Week 3 was introducing us to Internet cookies - these are all about maintaining state on a webpage.

Cookies are small pieces of information that are:

  • Sent from a website
  • Stored on the visitors hard drive by the user's web browser

Cookies are usesd to differentiate among users and personalise websites They are used to:

  • Remember stateful information (e.g. items added into a shopping cart), manage the shopping cart, personalise aspects of websites
  • Record the user's browsing activity (e.g. clicking particular buttons, logging in, or recording which pages were visited in the past) - website statistics and creation of anonymous user profiles
  • Remember arbitrary pieces of information previously entered (e.g. names, addresses, passwords, and credit card numbers) - knows if the user is already logged in

Week 4

Summary

Week 4 we began to look at the jQuery library, this is a fast and concise JavaScript Library that is designed to change the way that you write JavaScript

It offers some different functionality from plain JavaScript

  • DOM scripting & event handling
  • Ajax
  • User interface effects
  • Form validation
  • Utilities
It is lightweight, supports multiple browsers, has a CSS-like syntax, an active developer community supporting the library and is extensible.

jQuery can be downloaded from the website http://jquery.com/ or it can be linked to from external sources such as

  • https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js
  • http://code.jquery.com/jquery-3.3.1.min.js

Selecting a part of the document is a fundamental part of jQuery.
a jQuery object is a wrapper for a selected group of DOM nodes
$() function is a factory method that creates JQuery objects
$(a) is a JQuery object containing all the a elements in the document

$ is nothing but a shorthand notation for find method in JQuery. You can also use jQuery('' '') instead of $('' '')
