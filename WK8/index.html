Week 8

Summary

Week 8's lecture was titled form validation and covered how to:

  • Use form validation to examine text
  • Understand the princiuples of working with text fields
  • Detect and change the format of information in a text field
  • Work with radio buttons, check boxes, and selection menus
  • Create simple, self grading tests

There are different types of form validation, including: valid email address, valid date, valid text type and leaving required fields empty

We should remind the visitor to enter missing data and reformat the visitor's data as needed
